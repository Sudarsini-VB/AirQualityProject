# 🌍 AI Air Quality + BI Analyst Intelligence System

Live App: https://airqualityproject-ts4tpyltpkpsdhhemvuzuz.streamlit.app/

## Overview
AI-powered Streamlit app for AQI checking and end-to-end dataset analysis with automated insights, anomaly detection, and dashboards.

## Features
- AQI Checker with health advisory
- Dataset upload & full analysis
- Data profiling (missing, types, unique)
- KPI dashboard (avg, max, risk)
- Outlier & anomaly detection
- Trend analysis (rolling mean)
- Regional comparison (best/worst)
- Correlation matrix
- Download processed data

## Tech Stack
Python, Streamlit, Pandas, NumPy, Matplotlib

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
